# geany-setup
Geany setup for competitive programming

## Install on Ubuntu

    sudo apt install geany geany-plugins geany-plugin-vimode cppcheck -y
    cd ~/.config
    mv geany geany.bak
    git clone https://github.com/gabrc52/geany-setup.git geany
